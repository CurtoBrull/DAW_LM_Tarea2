FORMULARIOS

Los formularios en html se inician con la etiqueta <form> y cierran con </form>. En su interior pueden contener la etiqueta <label> </label> que nos permitirá pulsar en cualquier punto del texto si hemos rellenado el atributo id y empezar a rellenar el input correspondiente.

La etiqueta <input>, que no necesita cierre, nos permitirá definir junto con el atributo type que tipo de formulario queremos usar. Por ejemplo en la siguiente línea de formulario tenemos un input de tipo texto, otro de tipo date y de tipo time.
Si queremos introducir un texto de ejemplo necesitaremos el atributo placeholder. Un ejemplo completo de un imput puede ser el siguiente: <input type="text" placeholder="Tu nombre">

A continuación tenemos inputs de tipo week y month. Tambien se ha incluido un input de tipo submit que nos crea un botón con la función de enviar los datos introducidos.

Un formulario de tipo datetime-local nos puede resultar bastante completo para introducir una fecha completa y una hora.

Los siguientes inputs sirven para poder introducir de forma automática, según datos guardados anteriormente, correos electrónicos con el atributo autocomplete ="email", otro para el país con autocomplete ="country" y por último un código postal con autocomplete ="postal-code".

La etiqueta <datalist></datalist> nos permiten crear listas desplegables con valores introducidos mediante tantas etiquetas <option>/<option> como sean necesarias y el atributo value.

El input de tipo radio nos permite crear un conjunto de opciones seleccionables, en este caso solo permite seleccionar 1 de las opciones disponibles.

El input de tipo checkbox nos permite crear un conjunto de opciones seleccionables, en este caso solo permite una selección múltiple.

La etiqueta <textarea></textarea> nos permite crear un área donde introducir el texto a enviar. Podemos configurar la cantidad de columnas con el atributo cols y las filas con el atributo rows. Tambien podemos usar el atributo maxlength para limitar el número máximo de caracteres que se pueden introducir.


Otro tipo de formulario que podemos usar es el organizado en un grupo de campos. La etiqueta <fieldset></fieldset> nos permite contener los inputs que necesitemos.
